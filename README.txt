Всем привет! 
Это обучающий репозиторий, который будет использоваться для постижения основ GitHub
Если вы уже знакомы с гитом, то задания не вызовут у вас затруднений, иначе, ниже будет прикреплена ссылка.
По этой ссылке размещен очень хороший мануал по гиту и все, что нам потребуется, вы сможете узнать там.

В папке Tasks/ вы должны будете создать собственную папку с вашей фамилией (лучше транслитом)
В этой папке вы все будете размещать ваши файлы по заданиям

В любой момент вы можете написать мне, если возникли трудности :)

ЗАДАНИЕ 1
0) Установить и настроить терминальную версию git:
    > sudo apt-get install git // for Ubuntu
    > brew install git // for MacOS

    Если пока трудно, используйте браузерную версию)
1) Сделать Fork этого репозитория
2) В форке создать папку с вашей фамилией в папке Tasks/
3) В ней сделать один файл, названный FirstStep.txt
4) В этом файле должны быть:
    - Ваш ник на гитхабе
    - Опыт в разработке
    - Как вы знаете python? (По шкале от 0 до 10) + словесное описание вашего уровня
    - Есть ли Linux/MacOS?
    - Какие среды разработки у вас стоят?
    - Опыт работы в терминале
    - Знание баз данных и SQL
    - Опыт web разработки
    - Готовность обучаться летом
    - Ваши предпочтения (что бы хотелось прогать, чем бы зотелось заниматься)
    - Другое (здесь вы можете указать то, что считаете нужным)
5) После того, как вы создадите этот файл, вы должны послать pull request и написать + мне в личку

Удачи!
